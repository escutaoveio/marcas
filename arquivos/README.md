# Base de Conhecimento

Hub de referência global do ecossistema IVA Química. Alimenta briefings, campanhas, conteúdo e decisões estratégicas de todas as frentes.

---

## Subpastas

| Pasta | O que entra aqui |
|---|---|
| `catalogo/` | Produtos ativos: SKUs, descrições técnicas, tabelas de especificação, fotos de referência |
| `personas/` | Perfis de público: cliente final, profissional (pintor, marceneiro, decorador), B2B |
| `mercado/concorrentes/` | Fichas de concorrentes: posicionamento, canais, pontos fracos, ameaças |
| `mercado/tendencias/` | Tendências de mercado, materiais de construção, decoração, consumo |
| `mercado/pesquisas/` | Pesquisas qualitativas e quantitativas de público e mercado |

---

## Como Usar

**Antes de criar um briefing de campanha:** consulte `personas/` para confirmar o público e `catalogo/` para os produtos corretos.

**Antes de definir posicionamento:** consulte `mercado/concorrentes/` para não repetir o que a concorrência já faz.

**Para alimentar o `/intel`:** popule `mercado/tendencias/` e `mercado/pesquisas/` com dados recentes.

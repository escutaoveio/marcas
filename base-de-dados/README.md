# Base de Dados

Repositório central de dados operacionais da IVA Química. Planilhas, dashboards públicos e relatórios consolidados.

---

## Subpastas

| Pasta | O que entra aqui | Formato |
|---|---|---|
| `planilhas/` | Dados brutos e operacionais organizados por contexto | `.xlsx`, `.csv`, `.ods` |
| `dashboards/` | Visualizações interativas com acesso público via GitHub Pages | `.html` |
| `relatorios/` | Consolidações prontas para envio aos interessados | `.html`, `.pdf` |

---

## Distinção importante

| `base-de-dados/` | `arquivos/reports/` |
|---|---|
| Dados vivos, atualizados regularmente | Registros históricos fechados |
| Planilhas e dashboards interativos | Relatórios gerados por período |
| Fonte de verdade dos números | Saída final para distribuição |

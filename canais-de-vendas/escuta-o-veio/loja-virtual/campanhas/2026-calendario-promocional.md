## Como Usar Este Arquivo

1. Para cada campanha, crie uma subpasta em `canais-de-vendas/escuta-o-veio/loja-virtual/campanhas/`
2. Nomeie como: `AAAA-MM-DD-nome-da-campanha/`
3. Dentro, crie o arquivo de briefing usando `_template-campanha.md`
4. Atualize o status aqui conforme a campanha avança

# Escuta o Véio! — Loja Física

> Canal físico da marca Escuta o Véio!, extensão da operação de e-commerce para o varejo presencial.

Loja própria IVA Química. Integra com o e-commerce via OMS (estoque unificado) e Click & Collect.
